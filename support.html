Troubleshooting Guide

If you encounter problems, try the following steps before reaching out:

  • Ensure you are using the latest version of Chameleon Hub.
  • Run the application as an administrator to avoid permission issues.
  • Check if your antivirus or firewall is blocking the application.
  • Verify game file paths and ensure game files are not corrupted.

Frequently Asked Questions

Q: How do I install Chameleon Hub?
A: Download the latest release from the Download page and run the installer.

Q: Where can I report bugs?
A: Please use the GitHub Issues page on our repository.

Q: Chameleon Hub is not detecting my game files. What should I do?
A: Make sure your game directory is correctly selected in the application settings and that you have the correct file permissions.

Q: Can I load multiple mods at the same time?
A: Yes! Use the Mod Loader feature to manage and prioritize multiple mods. Check the Features page for more info.
